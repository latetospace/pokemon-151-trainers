# 151 Trainers

**151 Trainers** is a simple, static web project that displays the original 151 Pokémon as collectible cards, each associated with a trainer and a card count.

The site is fully client-side and powered by a single HTML file and a JSON data file, making it easy to maintain and deploy with GitHub Pages.

---

## Features

- One-page grid of all 151 original Pokémon
- Each Pokémon shows:
  - Sprite image
  - Pokédex number
  - Trainer name
  - Number of cards collected
  - Optional X (Twitter) profile link
- Fast search by Pokémon name or Pokédex number
- Data-driven via JSON (no backend required)
- If a Pokémon has no entry, it defaults to:
   - Trainer: Unclaimed
   - Cards Collected: 0
